# EVO TechCar IoT

EVO TechCar IoT is a Home Assistant integration that connects vehicles equipped with the EVO TechCar telematics system.

## Features

- Vehicle sensors (OBD, telemetry)
- GPS device tracker
- Relay control (PCF8574)
- Cloud API integration
- Secure authentication

## Requirements

- EVO TechCar account
- Home Assistant 2023.1+
